环境配置

  • 安装 Node.js。本人安装V6.9.1LTS。
  • 安装Git。我是在当年读《Android第一行代码》时候装的一个windows版本。
  • 安装hexo：npm install -g hexo。初始化（在指定文件夹，使用git bash）：hexo init。

hexo初步

初始化之后，我们就可以做很多事情了，先记住下面的命令：

hexo g#=hexo generate，生成
hexo s#=hexo sever，本地服务器
hexo d#=hexo deploy，同步github
hexo d -g#=hexo generate ,hexo deploy

先生成，后服务器运行，在浏览器输入http://localhost:4000/ 查看。注意，如果此刻你的文件夹没有.deploy_git文件夹，那么请自己百度下载方法，进行下载，而public文件夹会在hexo g后出现。

GitHub Pages

  • 新建仓库，取名：bestzem.github.io
  • 进入setting，在相应模块Launch。
  • 进入.deploy_git文件夹，如果此时仓库里面有git默认生成的文件，请clone一下，然后在.deploy_git文件夹进行取舍。
  • 下面，我们添加ssh。

添加ssh

先设置用户名和邮箱：

git config --global user.email "email"
git config --global user.name "name"

然后生成相关文件：

ssh-keygen -t rsa -C "email"

至此，请一路回车（如果你操作娴熟，了解原理，可用自己设置路径），输入密码（请记住）。而后，在.ssh文件夹打开.pub文件，复制内容。点开仓库的setting，选择deploy keys，然后add。将复制内容填入key，title随意，请一定勾上Allow write access（随后才会拥有操作权利）。
